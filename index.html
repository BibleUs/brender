brender 0.5 beta

Brender is a distributed rendering solution for the 3D animation suite Blender. In this page you can download the compressed master folder form our github repository. The application is under heavy development and refactoring at the moment, so beware: things are not clear and might not work correctly! We will post an update here as soon as contents look more mature. Thanks for dropping by.
